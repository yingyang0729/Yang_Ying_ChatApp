## History

1. Clone the repository to your local machine:
```
git clone https://github.com/your-username/live-chat-app.git
```

2. Install the dependencies:
```
cd to destination folder
npm install express
```

3. Start the server:
```
npm start
```

4. Open the app in your browser:
```
http://localhost:3000
```
